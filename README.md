## Overview

This repository contains a **data cleaning template** built in Python using pandas.
It demonstrates best practices in:

- Loading and inspecting raw datasets
- Cleaning numeric and monetary columns
- Handling missing values
- Removing duplicates
- Outlier detection and handling
- Creating derived/feature columns
- Final validation and exporting cleaned datasets

This template is designed to quickly prepare datasets for analysis, visualization, or modeling.
